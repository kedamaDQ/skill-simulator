# DQ10 Skill simulator by foresdon.jp

This is a skill point assigning simulator for "DRAGON QUEST X online". Application is running at [app.foresdon.jp](https://app.foresdon.jp/skill-simulator/).

## Dependency

- javaScript(ES6)
- React
- react-rudux
